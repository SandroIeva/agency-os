# Reel Scripter

Writes the full short-form script — the format you batch most. Climax-led opener, tight beats, visual cues, CTA. In your voice.

## When to run this

- Batching day — turning ideas into shootable Reel scripts
- A Reel idea needs the full script written
- Adapting a longer piece into short-form

## Inputs

Required:
- The Reel idea/topic

Reads if available:
- Voice doc
- An opener from The Bridge (use it verbatim if supplied)

## Output structure

```
# Reel Script — [topic]

**Length:** [30-60s]
**Format:** [Reel / Short / TikTok]

## Opener
[Climax-led, one line. From The Bridge if supplied.]
[Visual cue]

## Beat 1
[One line] [Visual cue]

## Beat 2
[One line] [Visual cue]

## Beat 3
[One line] [Visual cue]

## Beat 4 (optional)
[One line] [Visual cue]

## CTA
[One line] [Visual cue]

## On-screen text suggestions
[Key words/phrases to caption on screen]
```

## Process

1. Read voice doc
2. Use the Bridge opener if supplied, otherwise write a climax-led one
3. Write 4-5 beats max — short-form dies if overstuffed
4. Add visual cues per beat
5. Write the CTA
6. Suggest on-screen text

## Critical rules

- 4-5 beats MAXIMUM. More is too dense for short-form.
- Every beat is ONE line. If it needs two, it's two beats.
- Opener is climax-led. Use The Bridge output verbatim if provided.
- Visual cue per beat — the script must be shootable, not just readable.
- Match the voice doc. No AI tells.
- The CTA ties to the current goal where possible.
