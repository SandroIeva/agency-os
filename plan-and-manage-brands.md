# Plan and manage brands

### Projects

Create projects for brands, campaigns, launches, or internal work. Add a name, visual identity, and members so the project is easy to recognize.

### Kanban

Kanban provides a clear view of work as it moves through stages such as:

* To Do
* In Progress
* Review
* Done

Create a task with a title, description, priority, owner, project, and due date. Tasks can also contain:

* checklists
* comments
* attachments
* collaborator notifications

Drag a task between columns as its status changes.

### Timeline and sprints

Use Timeline for work that needs structure across a longer period. Group items into sprints, connect tasks, assign collaborators, and monitor progress across projects.

Timeline works well for:

* campaigns
* website or product launches
* brand rollouts
* recurring content programs
* client delivery phases

### Calendar

Calendar combines scheduled work with Google Calendar events. Create or update events, view upcoming commitments, and keep deadlines visible alongside project tasks.

### Notes and reminders

Capture information that does not need a full document or task. Reminders can also be synchronized with Google Calendar so they remain visible across devices.

### A simple planning routine

1. Create the project.
2. Define the next milestone or sprint.
3. Break it into tasks.
4. Assign owners and deadlines.
5. Use Review before marking work Done.
6. Check the timeline and calendar for conflicts.
