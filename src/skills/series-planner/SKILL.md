# Series Planner

One big idea becomes a connected series. Maps the parts across platforms so a single theme compounds instead of being spent in one post.

## When to run this

- A topic too big for one post
- Building a content arc that keeps the audience returning
- Planning a launch runway (series leading to an offer)
- Wanting a theme to span YouTube + Instagram + newsletter coherently

## Inputs

Required:
- The big topic or theme

Reads if available:
- Pillars, goal, the platforms in play

## Output structure

```
# Series — [theme]

## The arc
[The through-line — what the series builds toward across all parts]

## The parts

### Part 1: [Title] 
**Angle:** [The specific take]
**Platform:** [YouTube / Instagram / Newsletter]
**Format:** [Long-form / Reel / carousel / email]
**Connects to:** [How it sets up the next part]

### Part 2: ...
[Continue for 4-6 parts]

## The platform map
[Which parts live where, and how they cross-reference each other]

## The payoff
[Where the series lands — the offer, the realisation, the CTA it builds to]

## Start here
[The first part to make and publish]
```

## Process

1. Define the arc — what the whole series builds toward
2. Break into 4-6 parts, each with angle, platform, format
3. Show how each part connects to the next
4. Map across platforms
5. Define the payoff the series drives to
6. Name the starting part

## Critical rules

- The parts must CONNECT — a series is not just 5 posts on a topic, it's a sequence that builds.
- Map across platforms intentionally — YouTube depth, Instagram moments, newsletter nurture.
- The arc must build to a payoff — a realisation, an offer, a CTA. Series without destinations meander.
- 4-6 parts. Fewer isn't a series; more loses the audience.
- Name the starting part — the creator needs one clear first move.
